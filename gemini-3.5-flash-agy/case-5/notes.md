# Case 5: 睡眠监测桌面卡片 notes

## 1. 布局选型与设计考量 (Layout Design & Selection)

- **卡片尺寸选择**: `2x2` (160 x 160vp)。
- **选型理由**:
  - 本卡片主要用于展示单一核心健康指标（睡眠时长与睡眠状态圆环）并提供单一动作（早睡提醒），不需要复杂的横向多区域或长数据列表，符合 `2x2` “一个主要结论和一个辅助动作” 的卡片构造规则。
- **信息层级与视觉焦点**:
  - **顶部 (Status)**: “睡眠不足”状态提示，使用半透明红色背景小胶囊设计，既醒目又不过分喧宾夺主。
  - **中部 (Primary Answer / Metric)**: 视觉核心，使用 Stack 将睡眠时长文本（“6小时”、“45分”）重叠嵌入在睡眠状态进度圆环（Progress ring）中，一目了然且科技感十足。
  - **底部 (Action)**: “设定早睡提醒”按钮，带闹钟图标 (⏰)，支持快速点击交互。
- **硬预算分配 (Height & Width Budget)**:
  - 目标卡片大小：`160 x 160vp`，内部 Padding：`12vp`，可用内部区域：`136 x 136vp`。
  - 纵向高度预算：
    - 顶部状态栏: `18vp`
    - 中部进度环: `76vp`
    - 底部提醒按钮: `28vp`
    - 子节点间距 (itemMargin): `6vp` (共2个间距，共 `12vp`)
    - 总纵向占用：`18 + 76 + 28 + 12 = 134vp`。完美适配 `136vp` 高度，保证无垂直滚动。
  - 横向宽度预算:
    - 顶部“睡眠不足”胶囊：自适应包围文本，且在 Column 中居中对齐，不被挤压。
    - 中部圆环与文本：进度环为 `76vp` 宽，内部容纳两行 `12fp` 文本（“6小时”和“45分”），横向空间充足，避免了单行“6小时45分”在小环内的挤压。
    - 底部按钮: 宽度 `100%` (`136vp`)，`fontSize: 11`，文本 `⏰ 设定早睡提醒` 共7个字符，宽度约 `77vp`，预留足够的安全宽度以防文字溢出或截断。

## 2. 显式设计改进 (Design Improvements)

- **文本重构保护**: 初版考虑在环内单行显示“6小时45分”，但由于进度环内径较窄，容易导致内容截断或使用 Ellipsis 变成“6小时...”。改进方案为将时长拆为两行“6小时”与“45分”上下堆叠在环内，确保所有关键时长数据 100% 完整显示。
- **夜晚深邃色彩系统 (Color System)**:
  - 整体背景使用深紫色夜晚渐变，方向由左上到右下 (`RightBottom`)，色值从 `#0D071E` 渐变至 `#311E68`。
  - 睡眠不足提示前景色采用警告浅红 `#FF8A80`，辅以低透明度警告红背景 `#29FF5252`，起到警示作用且安静柔和。
  - 圆环采用极具科技感的淡紫色 `#A77DFF` 强调色；底部按钮使用靛紫色 `#5D3EBF` 背景，与主背景和谐共存又兼顾了点击引导。
